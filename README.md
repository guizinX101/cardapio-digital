# 🍔 Cardápio Digital Profissional

Sistema de cardápio online moderno com integração direta ao WhatsApp, ideal para lanchonetes, hamburguerias e deliverys.

---

## 🚀 Sobre o projeto

Este projeto foi desenvolvido para facilitar pedidos online sem a necessidade de aplicativos complexos ou taxas de marketplace.

O cliente acessa o cardápio, escolhe o produto e finaliza o pedido diretamente pelo WhatsApp com mensagem automática.

---

## ✨ Funcionalidades

* 📱 Layout 100% responsivo (mobile e desktop)
* 🛒 Listagem dinâmica de produtos
* 💬 Integração direta com WhatsApp
* ⚡ Carregamento rápido
* 🧩 Código simples e fácil de personalizar
* 🎯 Ideal para pequenos negócios

---

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript puro (sem frameworks)
